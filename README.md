# FerriteHyperelastic

A Julia Package for finite element anlysis of hyperelastic materials.

Built on top of https://ferrite-fem.github.io/Ferrite.jl/stable/

Supports:

> * Plane Stress

> * Plane Strain

> * 3D

> * curve fitting

To install

```
julia> ]

(@v1.11) pkg> add https://github.com/Aminofa70/FerriteHyperelastic.jl
````
