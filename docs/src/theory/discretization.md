## The discretization of residual 

The interpolation for the displacement at the element level is 

$$\mathbf{u} = \sum_{a=1}^{n_e} N_a \, \mathbf{d}_a$$

::: info
- $\mathbf{d}_a$: nodal displacement of node $a$  
- $n_e$: total number of nodes in element $e$  
- $N_a$: scalar shape function of node $a$
:::
